- calc模块可以替代le_trans，如：
calc
    x\le y := proof_1
    _\le z := proof_2
Lean会自动推断出 x\le z

- apply必须匹配等式**右边**。想要更改左边，使用rw。
- eg. 【theorem absorb1 : x ⊓ (x ⊔ y) = x := by
            rw[inf_sup_left x x y]】
    报错的原因是，这里是一般的lattice。而分配律并非对所有lattice成立。如：五元素菱形格（最大元+最小元+三个中间无关元素）
- inf的优先级高于sup的优先级。而且两者都是左结合的。
